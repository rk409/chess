# ♟️ Chess Game (JavaScript)

A simple chess game built using **HTML, CSS, and JavaScript**.

## 🚀 Features

- Fully functional chessboard
- Click & move pieces
- Visual feedback for invalid moves
- Responsive design

## ⚙️ How to Run

1. Download the project
2. Open `index.html` in your browser
